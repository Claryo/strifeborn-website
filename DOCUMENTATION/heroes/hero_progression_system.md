# Hero Level Progression (Out-of-Run)

## 1.1 Purpose & Design Intent

Hero Level Progression is a **long-term mastery system** that rewards commitment to individual heroes without undermining in-run decision making.

Hero Levels are designed to:

* Reinforce hero identity early
* Provide visible, understandable progression
* Reward long-term play without power spikes
* Add consistency and polish to hero performance

Hero Levels must **never**:

* Be required for a hero to be viable
* Replace in-run scaling systems (Tier, equipment, synergies)
* Create large power gaps between players
* Alter a hero’s role or counterplay

A low-level hero must always remain playable.  
A high-level hero should feel **more reliable and complete**, not mandatory.

## 1.2 Level Cap & Progression Scope

* **Maximum Hero Level:** 30
* Hero Levels persist across all game modes
* Levels are shared between PvE and PvP
* Level 30 represents **completion**, not infinite scaling

Hero XP is earned primarily by:

* Playing runs with the hero
* Completing encounters and waves
* Minor out-of-run activities (supplementary only)

Hero Levels **cannot** be purchased directly with currency.

## 1.3 Reward Philosophy

Hero Level rewards are divided into **four controlled categories**:

1. **Stat Rewards**
2. **Persistent Equipment Slots**
3. **Identity Milestones**
4. **Cosmetic Rewards**

Each category has a clear role and strict limits.

## 1.4 Stat Rewards Overview

Stat rewards represent **gradual mastery** and are intentionally small.

Their purpose is to make the hero feel:

“Slightly sharper, sturdier, or more reliable — but not stronger than the build.”

**Core Principles**

* All stat rewards are **flat**
* All stat rewards use **fixed values**
* Stat values **do not scale with level**
* Power comes from accumulation, not spikes

Hero Level stats are always applied to **base stats**, then scaled normally by Tier.

## 1.5 Primary vs Secondary Stats

### 1.5.1 Primary Stats (Role Definition)

Primary Stats define a hero’s **structural role**.

* Strength
* Wisdom
* HP
* Physical Defense
* Spell Defense

Primary stats are:

* Front-loaded early in progression
* Used sparingly later
* The majority of total stat rewards (~60%)

**Recommended Values**

|  |  |
| --- | --- |
| Stat | Value per Reward |
| Strength | +2 |
| Wisdom | +2 |
| HP | +25 |
| Physical Defense | +9 |
| Spell Defense | +9 |

### 1.5.2 Secondary Stats (Expression & Feel)

Secondary Stats refine **cadence, reliability, and comfort**.

* Crit Chance
* Crit Damage
* Attack Speed
* Move Speed
* Starting Mana

Secondary stats are :

* More common in mid and late progression
* Never role-defining
* Roughly 40% of stat rewards

**Recommended Values**

|  |  |
| --- | --- |
| Stat | Value per Reward |
| Crit Chance | +1% |
| Crit Damage | +5% |
| Attack Speed | +0.05 |
| Move Speed | +1 |
| Starting Mana | +5 |

## 1.6 Equipment Slots & Relic Unlock Timing

### 1.6.1 Persistent Equipment Slots

Equipment systems are unlocked **early** to avoid frustration.

|  |  |
| --- | --- |
| Hero Level | Unlock |
| 3 | Equipment Slot 1 |
| 6 | Equipment Slot 2 |
| 9 | Equipment Slot 3 |
| 12 | Equipment Slot 4 |
| 14 | Equipment Slot 5 |

All equipment slots are unlocked **before** the Hero Relic.

### 1.6.2 Hero Relic Unlock

* **Hero Level 15** unlocks the Hero Relic
* The relic is added to the run loot pool
* It is **not granted automatically**
* It does not bypass in-run economy rules

Relics extend hero identity but are never mandatory.

## 1.7 Identity Milestones

Each hero has **exactly three Identity levels**.

Identity rewards :

* Refine existing mechanics
* Improve reliability or thresholds
* Slightly extend scaling ceilings

Identity rewards must **never**:

* Add new mechanics
* Remove weaknesses
* Change the hero’s role
* Invalidate counterplay

Identity levels are fixed at :

* **Level 10**
* **Level 20**
* **Level 29**

These milestones are intentionally rare and impactful.

## 1.9 Cosmetic Reward Scope

Each hero has **five mastery cosmetics**:

* Base skin variant
* Title
* Banner
* Frame
* Prestige skin (Level 30)

Cosmetics are :

* Non-repeatable
* Never filler
* Never tied to power

## 1.10 Design Contract

Hero Level Progression is a **mastery layer**, not a balance lever.

If Hero Levels ever :

* Become mandatory
* Outscale in-run systems
* Create competitive imbalance

Then the system must be revised.
